## Search Form Features

### 1. Input Field
- **Type:** Material Form Field with outline appearance
- **Icon:** Search icon (ğŸ”) as prefix
- **Behavior:** Two-way binding with `ngModel`
- **Placeholder:** Descriptive text for each component

### 2. Search Button
- **Color:** Primary (blue)
- **Icon:** Search icon
- **Action:** Submits form and triggers search
- **Keyboard:** Can be triggered by Enter key

### 3. Clear Button
- **Color:** Default
- **Icon:** Clear/X icon
- **Action:** Clears search term and reloads all data
- **Type:** Button (doesn't submit form)

### 4. Form Behavior
- **Submit:** Applies filter using `FilterOperator.LIKE`
- **Clear:** Removes all filters
- **Empty Search:** Shows all results (no filter applied)
- **Validation:** Trims whitespace from search term
